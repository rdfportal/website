The wwPDB RDF dataset has been updated in RDF Portal.

This release includes minor changes to support extended PDB IDs, which are 12-character PDB identifiers such as pdb_000010fm. The wwPDB plans to transition to extended PDB IDs as the standard format in July 2027.

In wwPDB RDF, the existing RDF resources based on conventional four-character PDB IDs will continue to be maintained. For example, the URI based on the four-character PDB ID will remain unchanged:

```ttl
http://rdf.wwpdb.org/pdb/10FM
```

In this release, the corresponding extended PDB ID has been added as an alternative label, for example:


```ttl
skos:altLabel "pdb_000010fm"
```
