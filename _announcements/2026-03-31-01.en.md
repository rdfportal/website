## Released the redesigned RDF Portal website

We are pleased to announce the release of the redesigned RDF Portal website. This is the f
irst major redesign since the portal's initial launch in 2017.

The new website has been rebuilt with an improved structure and updated content to better 
serve the life sciences research community.

As part of this renewal, we have also undertaken a systematic curation of data schemas for
 all datasets hosted on the portal, based on the [RDF-config](https://github.com/dbcls/rdf
-config) framework. Each dataset now has an explicitly described, machine-readable structu
ral model, ensuring consistent documentation and enabling automated generation of schema diagrams and SPARQL query templates across the entire portal.

Key changes include:

### Improved dataset pages

- Each dataset now features detailed metadata, statistics, example SPARQL queries with direct execution links, and schema diagrams automatically generated from RDF-config.
- Datasets can be filtered by domain tags, provenance, and registration type.

### New access methods

- **MCP interface (TogoMCP)** — A Model Context Protocol server that enables AI assistants such as Claude, ChatGPT, and Gemini to access RDF Portal data through natural language. See the [MCP interface](https://rdfportal.github.io/website/access_methods/mcp_interface/) page and the [TogoMCP website](https://togomcp.rdfportal.org/) for details.
- **SPARQL composer** — A graphical interface for building SPARQL queries without writing syntax manually.

### New documentation

- **User manual** — A comprehensive manual covering all aspects of the portal, from getting started with RDF and SPARQL to advanced tutorials such as federated queries across multiple endpoints. Available in both English and Japanese.
- **Data submission guidelines** — Documentation for researchers who wish to submit their RDF datasets to the portal.
- **RDF-config documentation** — Detailed explanation of the RDF-config framework used to describe dataset structures across the portal.

### Other improvements

- Addition of an update log and an announcements page
- Redesigned site navigation and layout
- Bilingual support (English / Japanese) across all pages
- Updated download page with multiple RDF serialization formats

We will continue to expand the portal's content and features. For questions or feedback, please contact us through the [About](https://rdfportal.github.io/website/about/) page.
