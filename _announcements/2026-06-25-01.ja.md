wwPDB RDF を更新しました。

今回のリリースでは、pdb_000010fm のような12文字の PDB ID、すなわち extended PDB ID への軽微な対応がなされています。wwPDB では、2027年7月に extended PDB ID が標準形式へ移行する予定です。

wwPDB RDF では、従来の4文字 PDB ID に基づく RDF リソースは今後も維持されます。たとえば、4文字 PDB ID に基づく URI は、従来どおり以下の形式で提供されます。

```ttl
http://rdf.wwpdb.org/pdb/10FM
```

今回の更新では、対応する extended PDB ID を代替ラベルとして追加しました。たとえば、以下のような情報が追加されています。

```ttl
skos:altLabel "pdb_000010fm"
```
