Mouse Chromosome Ontology (MCO) has been newly added to RDF Portal.

MCO provides simple and stable URIs for semantically identifying mouse chromosomes in different mouse reference genome versions. It enables RDF datasets to describe chromosome-level genomic resources consistently across mouse reference genome builds and to link them to external resources.
